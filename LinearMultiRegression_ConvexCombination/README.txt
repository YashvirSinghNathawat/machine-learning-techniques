# Readme File

## Convex.ipynb and Regress.ipynb

This repository contains two Jupyter Notebook files: `Convex.ipynb` and `Regress.ipynb`, each implementing a different method.

### Instructions

1. **Setup**
   - Ensure you have Jupyter Notebook installed on your system.

2. **Running the Notebooks**
   - Open Jupyter Notebook.
   - Navigate to the directory containing the notebooks.
   - Click on `Convex.ipynb` to open it.
   - To run the cells, use Shift+Enter or click the "Run" button.
   - Repeat the above steps for `Regress.ipynb`.

3. **Output**
   - Notebooks will generate output based on provided input.

**Note:** Ensure all required input files are in the same directory as `Convex.ipynb` and `Regress.ipynb`.
